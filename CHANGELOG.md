## [1.0.0] — 2026-02-18

### Added

- Initial documentation repository with workflow guides, architecture overview, and security/privacy docs.
- Workflow guides for: Compress PDF, Compress to 1 MB, Compress for Email, Sign PDF, Sign Privately, OCR PDF, Make PDF Searchable.
- Architecture overview documenting the browser-first, stateless processing model.
- Security & privacy summary covering file handling and deletion policies.
- Performance guide covering speed factors, known limits, and quality vs compression trade-offs.
- FAQ, roadmap, and illustrative processing contracts.
- Sample placeholders and expected-output descriptions.
- SECURITY.md for vulnerability reporting.
- CODE_OF_CONDUCT.md and CONTRIBUTING.md for community guidelines.
