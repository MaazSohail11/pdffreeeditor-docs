# Assets

> This directory holds images and diagrams used in the documentation.

## Structure

```
assets/
├── images/
│   ├── compress-hero.png    ← Hero image for compression docs
│   ├── sign-hero.png        ← Hero image for signing docs
│   └── ocr-hero.png         ← Hero image for OCR docs
└── diagrams/
    └── system-overview.png  ← High-level architecture diagram
```

## Notes

- **Placeholder files** are included for the image/diagram slots. Replace them with actual screenshots or diagrams when ready.
- Recommended image dimensions: **1200 × 630 px** for hero images (matches OG image specs).
- Diagrams can be created with any tool (draw.io, Excalidraw, Figma) — export as PNG for compatibility.
