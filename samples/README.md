# Samples

> **No real documents are stored in this repository.**

This directory contains placeholder descriptions of example inputs and expected outputs to illustrate what the PDF Free Editor tools do. Actual PDFs are not included because:

1. We don't store user documents.
2. Sample PDFs would add unnecessary repository size.
3. You can test the tools directly at [pdffreeeditor.com](https://pdffreeeditor.com).

## Example Inputs

| File | Description |
|------|-------------|
| `sample-contract-placeholder.txt` | Description of a typical 3-page digital contract PDF (~200 KB) |
| `sample-scanned-placeholder.txt` | Description of a typical 5-page scanned receipt PDF (~8 MB) |

## Expected Outputs

See the `expected-output/` directory for descriptions of what the tools produce:

| File | Tool | What it shows |
|------|------|---------------|
| `compress-example.txt` | Compress PDF | Before/after file sizes |
| `ocr-example.txt` | OCR PDF | Searchable text layer added |
| `signed-pdf-example.txt` | Sign PDF | Signature placed on page |
